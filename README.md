# HPC Cluster Wiki

This repository contains documentation and guides for working with HPC (High-Performance Computing) clusters.

## Contents

### [Saga HPC Cluster Guide](saga.md)

A comprehensive guide for team members working on the Saga HPC cluster at SIGMA2/NRIS. This guide covers:

- SSH login and connection setup with 2FA
- Project directory structure and organization
- Conda environment configuration
- File transfer using rsync and SFTP
- Job submission with SLURM
- Resource monitoring (disk quota, CPU hours)
- Best practices for efficient cluster usage

The guide is based on the [official SIGMA2 documentation](https://documentation.sigma2.no/) and tailored for team workflows.

## About Saga

Saga is a supercomputer operated by SIGMA2/NRIS, located at NTNU in Trondheim, Norway. It provides approximately 140 million CPU hours per year and features:

- 364 compute nodes
- 16,064 CPU cores
- 32 NVIDIA GPUs (P100 and A100)
- 6.5 PB parallel file system capacity
